**6. Create `src-tauri/src/web_parser/parse.rs`:**
- `pub fn parse_build_response(resp: serde_json::Value, build_id: &str) -> Result<BuildPlan, ParserError>`
- Double deserialization:
  1. Extract `resp["data"]["response_data"]` as string
  2. Parse that string as JSON: `serde_json::from_str(&response_data_str)`
  3. Extract inner `data` object
- If inner data is null or missing `_id`, return `ParserError::BuildNotFound`
- Map fields:
  - `id` = `data["_id"]` as String
  - `char_class` = `data["char"]` as String
  - `title` = `data["title"]` as String (default to empty string if missing)
  - `variants` = `data["variants"]` array, map each to `Variant`
- For each variant:
  - `name` = variant["name"] as String (default empty)
  - `skill` = variant["skill"] as object, convert to `HashMap<u32, u32>`, **filter out entries where value == 0**
  - `skill_order` = variant["skillOrder"] as `Vec<u32>`
  - `equip_skills` = variant["equipSkills"] array, each `{ key, mods, rank }` maps directly to EquipSkill
  - `paragon` = variant["paragon"] as object (HashMap of board_name -> board_data), convert to `Vec<ParagonBoard>`:
    - For each (name, board_data): `ParagonBoard { name, index: board_data["index"], rotate: board_data["rotate"], nodes: board_data["data"] as Vec<String>, glyph: extract_glyph(board_data["glyph"]) }`
    - **Sort paragon Vec by `index` field ascending**
    - Glyph extraction: API returns `{"0": "glyph_name"}` object — extract first value as `Option<String>`. If object is empty or null, return None.

```rust
#[tauri::command]
async fn parse_build_link(
    url: String,
    state: tauri::State<'_, Mutex<AppState>>,
) -> Result<BuildPlan, String> {
    let client = web_parser::D2CoreClient::new();
    let build_plan = client.fetch_build(&url).await.map_err(|e| e.to_string())?;

    // Store in AppState
    let mut s = state.lock().unwrap();
    s.build_plan = Some(build_plan.clone());

    Ok(build_plan)
}
```
