# Milestone v1.0: MVP

**Status:** SHIPPED 2026-03-16
**Phases:** 1-6
**Total Plans:** 15

## Overview

Windows desktop tool that takes a d2core.com build link, parses it, and applies skills + paragon selections to a Diablo IV character via safe UI automation. Includes CI/CD for automated checks and release builds.

## Milestone Summary

**Key Decisions:**
- Rust + Tauri over Electron (smaller binary, Rust safety)
- UI automation over memory injection (anti-cheat safe)
- Direct TCB API over DOM scraping (no headless browser)
- Calibration wizard with 5-point coordinate capture
- cfg(windows) guards for cross-platform compilation

**Issues Resolved:**
- Double-serialized JSON format for TCB API wire protocol
- DPI scaling normalization across display configurations
- Resolution-adaptive coordinate math with 1080p baseline

**Technical Debt Incurred:**
- Paragon board pixel coordinates need empirical measurement per resolution
- Code signing not yet configured for release workflow
